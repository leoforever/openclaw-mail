# 常见邮箱 IMAP/SMTP 配置

## 配置方法

### 方式 1：使用 setup 向导（推荐）
```bash
cd ~/.openclaw/skills/mail
python3 scripts/setup.py
```

### 方式 2：手动编辑配置文件

**凭证文件：** `~/.openclaw/secrets/mail_creds`
```bash
MAIL_SMTP_HOST=smtp.example.com
MAIL_IMAP_HOST=imap.example.com
MAIL_USER=your_email@example.com
MAIL_APP_KEY=your_app_password
```

**配置文件：** `~/.openclaw/config/mail-client/config.json`
```json
{
  "smtp_port": 465,
  "imap_port": 993,
  "mail_from": "your_email@example.com",
  "default_folder": "INBOX",
  "max_results": 20,
  "allow_send": true,
  "allow_read": true,
  "allow_search": true,
  "allow_delete": false
}
```

## 配置步骤（以 QQ 邮箱为例）

### 1️⃣ 在 QQ 邮箱开启服务
1. 登录 QQ 邮箱网页版
2. 设置 → 账户
3. 开启 "POP3/SMTP 服务" 和 "IMAP/SMTP 服务"
4. 生成授权码（会发送到手机验证）

### 2️⃣ 备份当前配置
```bash
cp ~/.openclaw/secrets/mail_creds ~/.openclaw/secrets/mail_creds.bak
cp ~/.openclaw/config/mail-client/config.json ~/.openclaw/config/mail-client/config.json.bak
```

### 3️⃣ 修改凭证文件
```bash
cat > ~/.openclaw/secrets/mail_creds << EOF
MAIL_SMTP_HOST=smtp.qq.com
MAIL_IMAP_HOST=imap.qq.com
MAIL_USER=yourname@qq.com
MAIL_APP_KEY=your_auth_code_here
EOF
```

### 4️⃣ 修改配置文件
```bash
cat > ~/.openclaw/config/mail-client/config.json << EOF
{
  "smtp_port": 465,
  "imap_port": 993,
  "mail_from": "yourname@qq.com",
  "default_folder": "INBOX",
  "max_results": 20,
  "allow_send": true,
  "allow_read": true,
  "allow_search": true,
  "allow_delete": false
}
EOF
```

### 5️⃣ 测试配置
```bash
cd ~/.openclaw/skills/mail
python3 scripts/mail.py config
python3 scripts/init.py
```

## 安全提示

⚠️ **重要：**
- 使用**应用密码/授权码**，不要使用登录密码
- 凭证文件权限必须是 `600`
- 不要将凭证文件提交到 Git
- 定期更换应用密码

```bash
# 设置正确的权限
chmod 600 ~/.openclaw/secrets/mail_creds
```

## 多邮箱配置

如果需要配置多个邮箱，可以：

### 方式 1：切换配置文件
```bash
# 新浪邮箱
cp mail_creds_sina mail_creds

# QQ 邮箱
cp mail_creds_qq mail_creds
```

### 方式 2：使用环境变量
```bash
export MAIL_USER="user@qq.com"
export MAIL_APP_KEY="qq_auth_code"
export MAIL_SMTP_HOST="smtp.qq.com"
export MAIL_IMAP_HOST="imap.qq.com"
```

## 故障排查

### 测试连接
```bash
cd ~/.openclaw/skills/mail
python3 scripts/init.py
```

### 常见错误
- **认证失败**：检查应用密码是否正确
- **连接超时**：检查防火墙/网络
- **端口错误**：确认使用正确的端口（SSL: 465/993, TLS: 587）
- **权限错误**：`chmod 600 ~/.openclaw/secrets/mail_creds`
